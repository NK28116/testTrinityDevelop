# lib ディレクトリ

ビジネスロジック、API、ユーティリティ関数が格納されています。

## ファイル構成

### api.ts
APIクライアント関数群。
- RESTful APIとの通信処理
- HTTPリクエスト/レスポンスのハンドリング
- エラーハンドリングとリトライ機能

### testRunner.ts
テスト実行処理。
- テストケースの実行
- 動的モジュールインポート
- アサーション処理
- テスト結果の生成

### reportGenerator.ts
レポート生成処理。
- テスト結果の集計
- カバレッジレポートの生成
- レポート形式の変換

### utils.ts
ユーティリティ関数。
- 日付フォーマット
- データ変換
- バリデーション関数
- その他共通処理

## 設計原則

- 純粋関数の優先
- エラーハンドリングの統一
- 型安全性の確保
- テスタブルな設計
- 単一責任の原則
