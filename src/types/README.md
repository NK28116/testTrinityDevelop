# types ディレクトリ

TypeScript型定義ファイルが格納されています。

## 型定義ファイル一覧

### testcase.ts
テストケースに関連する型定義。
- `TestCase`: テストケースの基本情報
- `AssertionType`: アサーション種別の列挙型
- テストケース関連のユーティリティ型

### testresult.ts
テスト結果に関連する型定義。
- `TestResult`: テスト実行結果の詳細情報
- `TestStatus`: テスト状態の列挙型
- テスト結果関連のユーティリティ型

### report.ts
レポートに関連する型定義。
- `CoverageReport`: カバレッジレポートの構造
- レポート生成関連の型

## 型設計方針

- 厳密な型定義によるバグの事前防止
- インターフェースの一貫性確保
- APIレスポンスとの型整合性
- 拡張性を考慮した設計
- ドメインモデルの明確化

## 命名規則

- インターフェース: PascalCase
- 列挙型: PascalCase
- プロパティ: camelCase
- オプショナルプロパティには`?`を明記
